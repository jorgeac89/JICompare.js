##Examples

###Comparing canvas

```js
var canvas1 = document.getElementById("canvas1");
var canvas2 = document.getElementById("canvas2");
JICompare.Compare(canvas1, canvas2);
```

###Comparing chunks

```js
var canvas1 = document.getElementById("canvas1");
var canvas2 = document.getElementById("canvas2");
JICompare.CompareChunks(canvas1, canvas2, 2, 2);
```
